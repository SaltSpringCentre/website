How This Site Works

Site Architecture

This website is a collection of static HTML pages hosted for free on GitHub Pages. There is no WordPress, no database, no server to maintain.

Content is managed through external tools: Google Sheets, Google Calendar, Retreat Guru, and Square. When you update content in those tools, the website updates automatically on the next page load.

Retreat Guru

Programs & Registration

Connected

Programs, retreats, and registration are managed through Retreat Guru. The website automatically displays current programs using the Retreat Guru widget. When you add or update a program in Retreat Guru, it appears on the website automatically.

The widget appears on the Offerings page and Calendar page. Registration and payments are handled entirely by Retreat Guru.

Retreat Guru Dashboard Retreat Guru Help Center

How It Works

  • Programs are displayed via the "rbg-connect" JavaScript widget
  • No WordPress plugin is needed
  • The widget reads directly from your Retreat Guru account
  • Styling is partially customizable (colors and fonts are set to match the site)
Square

Payments & Donations

Setup Required

Donations and shop purchases are processed through Square. Square Payment Links let you create a checkout link in the Square Dashboard and paste it into the site. No coding needed to add new products or change prices.

Square Dashboard Create a Payment Link

Setup Steps

  1. Log into the existing Square account at squareup.com/dashboard
  2. Go to Payments > Payment Links
  3. Create a "Donation" link with preset amounts ($25, $50, $108, $200) and custom amount enabled
  4. Create individual links for each shop product (books, apparel, gift certificates)
  5. Copy each Payment Link URL and send to the site developer to wire into the donate buttons and shop cards
Tip: Square also offers an "Add button to website" option when creating a Payment Link. This generates a Buy Button embed code you can paste directly into the HTML.

Current Status

  • Donate modal shows tiers but the button links to a placeholder URL
  • Shop products display but purchase buttons are placeholders
  • Interac E-Transfer instructions are live (payment@saltspringcentre.com)

Fees

  • 3.3% + $0.30 per online transaction (Square free plan)
  • No monthly fee
  • All transactions visible in Square Dashboard alongside in-person sales
Google Forms

Contact & Request Forms

Setup Required

Contact form and other request forms will use Google Forms. Create a form in Google Drive, copy the embed code or link, and it gets added to the appropriate page.

Google Forms

Forms Needed

  • Contact / General Inquiry (for contact.html)
  • Accident / Incident Report
  • Schedule Adjustment Request
  • Time Away Request
  • Purchase Order
  • Guest Request forms
  • Other forms listed on the Community Dashboard

Setup Steps

  1. Create each form in Google Forms
  2. Set up email notifications so responses go to the right person
  3. Send the form URLs to the site developer to add to the dashboard and contact page

ACYR Schedule

Retreat Schedule Editor

Connected

The ACYR schedule is edited using a custom browser-based editor. Multiple people can edit simultaneously (presence tracking via heartbeat). Changes are synced through Google Apps Script. The public schedule is viewed at the "baked" URL with ?view appended.

Open Schedule Editor View Public Schedule

How to Use

  • Double-click a time slot to add an event
  • Click an event to edit title, venue, or duration
  • Drag events to move them, drag the bottom edge to resize
  • Click Publish to share your changes with other editors
  • Click Get Latest to load changes others have published
  • Export PDF for printing
WordPress

Connected

Blog posts are managed in WordPress. The blog page on this site fetches posts automatically from the WordPress REST API. To publish a new post, log into WordPress and create a post as usual. Posts appear on the site on the next page load.
